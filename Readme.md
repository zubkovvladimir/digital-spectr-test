<h2 align="center">Установка и запуск</h2>

### Скачать

```
git clone git@github.com:zubkovvladimir/digital-spectr-test.git
```

### Перейти в папку

```
cd digital-spectr-test
```

### Установить

```
npm install
```

### Запустить тесты

```
npm run test
```

### Собрать проект

```
npm run build
```

### Запустить

```
npm run start
```
